02 — PERSONAL PROJECT

개인 작업물

자체 IP — 엘리트 몬스터(Elite Monster) 기획 자료입니다.

ORIGINAL IP · MMORPG

엘리트 몬스터 Elite Monster

소환수와 플레이어 캐릭터의 연계를 핵심으로 하는 MMORPG 자체 기획 프로젝트입니다. 세계관 · 전투 · 캐릭터 · 스킬 · 아이템 · 퀘스트 · 거래 · UI 등 게임 전 영역에 걸쳐 시스템을 직접 설계했으며, 시뮬레이터를 통한 밸런스 검증까지 진행했습니다.

핵심 컨셉 요약

전투 컨셉

  • 플레이어 캐릭터(PC)와 소환수의 연계 전투를 핵심으로 한다
  • 유저가 소환수의 AI를 직접 제어하는 커맨드 시스템
  • 소환수 스킬 우선순위를 세팅 → 자동 시전, 필살기는 액티브로만 사용
  • 세팅 난이도: 디지몬 마스터즈 < 엘리트 몬스터 < 테라 수준

6개 클래스

  • 드래곤나이트 · 레인저 · 기갑슈터 · 환술사 · 팔라딘 · 검투사

5속성 상성 시스템

  • 노멀 · 물 · 불 · 바람 · 전기 — 먹고 먹히는 관계 형성
  • 속성 우세 시 1.5배 대미지 적용
  • 노멀 속성은 독립 속성으로 상성에 영향받지 않음

성장 차별화 요소

  • 유대감 시스템 — 소환수와 함께 전투 / 교감 시 누적, 진화 조건이 됨
  • 진화 시스템 — 최대 레벨 달성 시 엘리트 등급으로 상위 진화
  • 장비 강화 요소를 배제하여 캐릭터 간 격차 최소화 — 레벨과 세팅 중심의 성장

기획서 목차
